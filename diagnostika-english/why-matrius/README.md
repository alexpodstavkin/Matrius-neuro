# Изображения для блока WhyMatrius

Положите сюда 6 файлов **формата 4:3** (рекомендуем 800×600 или 1200×900, JPG):

| Файл | Что показать |
|---|---|
| `platform.jpg` | Скриншот игровой платформы Matrius — тренажёр или экран с персонажем |
| `stats.jpg` | Стат-плашка «7+ лет на рынке, 50 000+ учеников» (можно текстовый баннер или фото ребёнка за компьютером) |
| `motivation.jpg` | Кадр с поддержкой мотивации — ребёнок улыбается, бейджи/звёзды на платформе |
| `teachers.jpg` | Фото преподавателя или группы преподавателей |
| `license.jpg` | Лицензия Сколково или скан официального документа |
| `series.jpg` | Кадр из мультсериала Matrius |

**Источник:** забрать с [web.matrius.online/matrius-summer-school](https://web.matrius.online/matrius-summer-school/) или из общих ассетов Matrius.

Если файл отсутствует — карточка покажет плейсхолдер с логотипом школы (см. `WhyMatrius.tsx`, обработчик `onError`).
